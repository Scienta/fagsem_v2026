# Program fagseminar 

## TLDR; 
```text
WIFI: scandic_easy
```text

Målet med denne helgen er å ekserimentere med språkmodeller. Under seminaret skal vi skrive kode, kanskje gjøre noen administrative oppgaver og skrive litt tekst. Poenget er å skrive minst mulig direkte, OG MEST MULIG VED HJELP AV SPRÅKMODELL.

Streik: Foreløpig ser det ut til at arrangementet går som planlagt, til tross for streik! I verste fall har vi en backup-plan :-) 

### FØR seminaret må du: 
1) Installere egnet AI-verktøy du ønsker å bruke (Claude code, codex, open code, claude cowork, osv). 
2) Ha tilgang til Scienta github 
Spør tobiast om du lurer på noe eller trenger tilgang. 

Hvis du vil bruke Claude Code, finnes det oppsettsskript og veiledning her:
[Setup av Claude Code](./setup/README.md)

### PÅ fredag 24/4 må du: 
- Møte opp kl 12:00 på https://www.scandichotels.com/no/hotell/scandic-holmenkollen-park. Lunsj i resturanten på bord reservert Scienta. 
- Etter lunsj går vi til møterommet. Sett deg sammen med gruppa di (info lengre ned)
