# Kravspekk/prompt kokebok

Lag en kokebok web applikasjon som lagrer og vedlikeholder oppskrifter på matretter.
Den må ha tydelige seksjoner for ingredienser, fremgangsmåte og estimert tidsbruk. Ha med bilder.
Oppskriftene må kunne redigeres. Må kunne søke oppskrifter blant annet på smaker, kjøkken (ex kinesisk) og ingredienser
Legg til kategori (forrett, hovedrett++) på hver oppskrift så man kan sette sammen en meny med flere retter.

Teknologi:
Kokebok appen skal realiseres som en frontend og backend applikasjon. 
Backend skal være en java applikasjon med MySQL database og REST CRUD APIer.

Frontend skal kjøre i nyeste React rammeverk.
