# Tema 1 – Utvikler + agent i praksis
Hvordan jobber vi best sammen med en AI-agent i det daglige utviklingsarbeidet? Hvilke beskrankninger må vi sette for agentene og hvilke teknikker funker? Sub-agenter, agents.md, commands. Hvordan kan det integreres med resten av verdikjeden for programmering - CI/CD, verktøy for statisk kodeanalyse, linting, testdekning osv.

# Tema 2 – AI-assistert systemutvikling
Hvordan bør en AI-assistert utviklingsflyt se ut hvis den skal fungere sammen med behovsanalyse, brukerreiser, spesifikasjon, test og review? Hvordan påvirker det softwareutvikling i stort. Oppstår nye flaskehalser? Hvordan kan eventuelt de løses og AI-assisteres.

# Tema 3 – Lokal LLM i praksis
Når fungerer lokal LLM godt nok, og hva vinner eller taper vi sammenlignet med skybaserte modeller? Hvilke lokale modeller duger og hva krever de av maskinvare? Kan de brukes med agenter?

# Tema 4 – Flere parallelle kodeagenter
Hvordan kan flere kodeagenter samarbeide om utvikling av et system eller en større feature? Hvordan kan man bruke kodeagenter til å analysere og detaljere løsninger som igjen implementeres av andre agenter?

# Tema 5 - Personlig assistent – Automatisering av daglige oppgaver
Hvordan kan man bruke en språkmodell til å automatisere arbeid som ikke handler om programmering? - Typiske arbeidsoppgaver:  Bistå med funksjonell analyse,  prosjektledelse, administrative oppgaver osv.
